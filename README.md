# Jot
Jot is a lightweight text editor that lives in a single HTML file. It offers simple formatting, indented lists, continuously backs up your work in localStorage and exports itself into every document you save.

- Save any Jot document as HTML to export a local file. You can continue work on the exported file, but it no longer backs up automatically.

- You can bold text, italicize or underline to highlight it.

- Select text and paste an URL to create a hyperlink.  

- Type -, + or > at the very beginning of a line to create a list item or change its bullet style.
  - Swipe right or press Tab to increase a list item's indentation level.
    - Like so.
  - Swipe left or press Shift+Tab to decrease the indentation level.

Type Backspace or Enter at the very beginning of a list item to turn it into a regular line of text.

Swipe or use Tab in regular (non-list) lines to create simple tables.

Every new jot is automatically backed up in your browser's local storage.
- Swipe right from the left edge of the screen or press Escape to access an overview.
  - Completely empty jots disappear from the list.
- Use Reset to clear all entries.
- Use Recovery to access snapshots of the ten last jots you re-opened, in case there's an accident.

Pro tip: Try pasting images or web content into your jot 🪄
